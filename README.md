# 🐍 Dreamer Snake AI v3.1

基于强化学习与环境模型（World Model）构建的智能贪吃蛇代理。本项目采用类似 Dreamer 的架构（RSSM + Actor-Critic），并在 PyTorch 下实现。不仅拥有智能的自主学习策略，还具备极佳的 PyGame 视觉反馈系统。

## ✨ 核心特性

- **🧠 先进的 RL 架构**：内置基于状态空间模型（RSSM）的世界模型，能够预测环境动态。结合 Encoder、Decoder、Reward Model、Value Model 与 Actor 网络实现闭环训练。
- **🎯 目标软更新 (Soft Update)**：引入目标价值网络，使用指数移动平均（EMA）平滑 TD 目标，显著提升训练稳定性。
- **🎮 炫酷的视觉表现**：
  - 流畅的蛇身渐变色绘制。
  - 吃到食物后的动态粒子爆炸特效。
  - 食物自发光呼吸灯效果。
  - 实时的数据统计面板（Loss、Epsilon、得分等）。
- **⚙️ 动态控制台**：在训练过程中支持实时暂停、调整渲染速度，并支持一键手动保存权重。

## 🛠️ 环境依赖

在运行本项目之前，请确保您的系统中已安装 Python 3.8+，并安装以下依赖包：

```bash
pip install torch torchvision pygame numpy
```
*(注：如果您的设备支持 GPU，建议安装对应 CUDA 版本的 PyTorch 以获得最佳训练速度)*

## 🚀 快速开始

克隆本仓库或下载源码后，直接运行主程序即可开启训练与渲染：

```bash
python dreamer_snake.py
```

### ⌨️ 运行控制快捷键

在 PyGame 渲染窗口处于活动状态时，您可以使用以下快捷键实时干预训练过程：

- `[Space] (空格键)`：暂停/继续 训练与渲染。
- `[↑] (向上方向键)`：增加渲染与训练速度（最高加速 x10）。
- `[↓] (向下方向键)`：降低渲染与训练速度。
- `[S] 键`：手动触发 Checkpoint 保存（保存模型权重、经验池与训练统计数据）。

## 📂 存档与恢复 (Checkpoint)

系统默认每 10 个 Episode 会自动保存一次进度。存档文件默认存放在项目目录下的 `dreamer_snake_ckpt/` 文件夹中，包含：
- `model.pth`：所有神经网络的权重与优化器状态。
- `memory.pkl`：最近的经验回放池数据。
- `stats.pkl`：历史得分、奖励与 Loss 曲线统计。

下次启动程序时，系统会自动检测该文件夹并**无缝恢复上一轮的训练进度**。

## 🧠 模型架构简述

本项目深度精简了世界模型的架构，使其能够高效适配轻量级的贪吃蛇环境：
1. **Encoder**: 将 12 维的物理观测状态（食物方位、墙壁距离、危险探测等）映射为包含均值与标准差的潜在正态分布特征。
2. **RSSM**: 核心环境动态模型，接收当前的隐状态与动作，预测下一时刻的隐状态。
3. **Reward / Value Model**: 评估隐状态的即时奖励与长期期望价值。
4. **Actor**: 策略网络，基于隐状态输出动作的概率分布。
5. **Decoder**: 用于重构损失计算，辅助 Encoder 学习到最具代表性的特征表达。
